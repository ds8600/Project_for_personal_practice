# Lost Ark Market/Avatar Viewer (연습용)

## 목적
Lost Ark 오픈 API를 학습하기 위한 개인 프로젝트입니다. 아바타/거래소 검색을 제공합니다.

## 사용 API
- Markets: 아바타 검색

## 동작 방식
- 서버/클라이언트에서 공식 Open API를 호출
- 캐시: 1~10분 단위로 서버 캐시 적용 (Rate Limit 준수)
- 개인정보 미수집, 로그에는 요청 시각/엔드포인트 정도만 기록

## 공개 데모/문서
- (선택) GitHub Pages 또는 Notion 링크

## 문의

- ## 개인정보 처리
본 프로젝트는 사용자의 개인정보를 별도로 수집/저장하지 않습니다. 
요청 로그에는 요청 시각/엔드포인트 정도의 최소 정보만 남습니다.

## 이용 약관 준수
본 프로젝트는 Lost Ark Open API 이용 약관 및 Rate Limit을 준수합니다.
